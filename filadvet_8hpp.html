FilaDVet* criar_fila_d ( )

Cria dinamicamente uma fila vazia e retorna o endereço onde ela se encontra alocada.

Exemplo de uso:

#include<cstdlib>
#include "filadvet.hpp"
int main() {
FilaDVet *f = criar_fila_d();
return EXIT_SUCCESS;
}
Retorna
O endereço de memória da fila alocada dinamicamente.
